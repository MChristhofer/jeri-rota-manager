# JeriRota Manager

Sistema web desenvolvido para gestão de reservas, operação turística, pagamentos e prestação de contas entre agências parceiras.

O projeto foi criado com base em uma necessidade real da operação da Jeri Rota, permitindo controlar reservas, valores pagos, saldos no embarque, NET de serviços e repasses entre parceiros.

## Acesse o projeto

Sistema publicado no GitHub Pages:

https://mchristhofer.github.io/jeri-rota-manager/

## Funcionalidades

- Cadastro de reservas e clientes
- Controle de valor total da reserva
- Registro de sinal recebido
- Cálculo automático do saldo no embarque
- Status da reserva
- Status do pagamento
- Painel financeiro
- Página de operação
- Controle de prestação de contas por empresa parceira
- Registro de NET da reserva
- Controle de repasses entre agências
- Armazenamento local usando LocalStorage

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage
- GitHub Pages

## Objetivo do projeto

O objetivo do JeriRota Manager é organizar a rotina operacional de uma agência de turismo, substituindo controles manuais por uma aplicação simples, visual e funcional.

Além disso, o projeto serve como portfólio prático para demonstrar conhecimentos em desenvolvimento web, lógica de programação, organização de dados e regras de negócio.

## Regras de negócio aplicadas

O sistema considera situações reais da operação turística, como:

- Reserva confirmada com pagamento parcial
- Cliente com saldo para pagar no embarque
- Valor NET mínimo para execução do serviço
- Serviços vendidos por uma agência e executados por outra
- Repasses pendentes entre empresas parceiras
- Prestação de contas futura entre agências
